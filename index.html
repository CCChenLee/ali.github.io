N16班的约法三章

一.考勤方面

  1. 不要迟到早退。
  2. 不要随便请假。
  3. 没来上课的同学要提前请假并写假条。
  4. 请假超过一天，则需要让家长打电话请假。

电话号码：131-1256-1886

二. 卫生方面

  1. 关注自身仪容仪表，轮流值日打扫教室。
  2. 早上点名后不准在教室里吃早餐。
  3. 下课离开教室前把自己座位上没用的东西随手扔到门外的垃圾桶里。
  4. 每天最后离开教室的同学要随手关电关门。

三. 课堂表现

  1. 上课认真积极听讲。
  2. 有布置作业要按时完成。
  3. 有疑问要及时向老师反馈。
  4. 踊跃配合晓盛老师的课堂互动。

最后祝愿大家在晓盛老师的带领下，学有所成，共同进步！
